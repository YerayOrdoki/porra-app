# Porra App

Proyecto base en React + Vite + Supabase para una app móvil/PWA de porra entre amigos.

## Arranque

1. Copia `.env.example` a `.env`
2. Rellena tus credenciales de Supabase
3. Instala dependencias:
   npm install
4. Ejecuta en desarrollo:
   npm run dev

## Incluye

- Auth con Supabase
- Navegación móvil inferior
- Pantallas: Inicio, Mi porra, Clasificación y Perfil
- Validación de duplicado preparada para capturar el error del índice único del backend
